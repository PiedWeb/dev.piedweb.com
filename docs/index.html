How to contribute ?

Contributions are welcome.
You just need to send a Pull-request via Github for the concerned package.

Before to push, check :

  • Coding Standards (vendor/bin/phpcs --standard=psr2 src/).
    I recommend to use php-cs-fixer configure as :
    php-cs-fixer fix src --rules='{"@Symfony":true, "array_syntax": {"syntax": "short"}}' --verbose
  • Test the code you just add and check if (phpunit) everything is working (composer test)
  • Realease follow semver form 0.1.0.
  • Try to keep commit message consistent. Don't hesite to rewrite the history.
